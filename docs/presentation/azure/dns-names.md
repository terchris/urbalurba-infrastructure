# DNS Names

The API system has the following dns names:

For production

* https://api.redcross.no - API endpoint URL for production
* https://developer.redcross.no - Developer API portal

For test

* https://api-test.redcross.no - API endpoint URL for test
* https://developer-test.redcross.no - Developer API portal for test


For Development

* https://api-dev.redcross.no - API endpoint URL for development
* https://developer-dev.redcross.no - Developer API portal for development
