# DNS Names

The API system has the following dns names:

For production

* https://api.urbalurba.no - API endpoint URL for production
* https://developer.urbalurba.no - Developer API portal

For test

* https://api-test.urbalurba.no - API endpoint URL for test
* https://developer-test.urbalurba.no - Developer API portal for test


For Development

* https://api-dev.urbalurba.no - API endpoint URL for development
* https://developer-dev.urbalurba.no - Developer API portal for development
